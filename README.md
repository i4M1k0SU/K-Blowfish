# K-Blowfish

Reverse engineered Blowfish implemented by K\*\*\*\*\*.

Reimplemented in C++.

## Usage

### Decrypt

```cmd
 K-Blowfish.exe --iv <IV Hex> --key <Key Hex> INPUT_FILE OUTPUT_FILE
```

### Encrypt

```cmd
 K-Blowfish.exe --iv <IV Hex> --key <Key Hex> --encrypt INPUT_FILE OUTPUT_FILE
```
